# 🏥 HemoVision - نظام كشف الأنيميا بالذكاء الاصطناعي

## 📋 نظرة عامة
HemoVision هو تطبيق ويب ذكي للكشف المبكر عن الأنيميا باستخدام صور العين والأظافر، مع دعم إدارة المرضى ونصائح التغذية.

## ✨ المميزات الرئيسية

### 1. 🤖 نموذج ذكاء اصطناعي حقيقي
- **الخوارزمية**: XGBoost (Extreme Gradient Boosting)
- **الدقة**: ±1.48 g/dL معدل خطأ مطلق
- **البيانات**: مدرب على 93 مريض حقيقي من الهند
- **الميزات**: 34 ميزة ملونة مستخرجة من كل صورة

### 2. 👁️ تحليل صور العين
- تحليل لون الملتحمة (Conjunctiva)
- قياس مؤشر الشحوب (Pallor Index)
- تقدير الهيموجلوبين، RBC، MCV

### 3. 💅 تحليل صور الأظافر
- تحليل لون فراش الأظافر
- قياس وقت الامتلاء الشعري
- درجة لون الأظافر

### 4. 🔐 نظام مستخدمين محمي
- Flask-Login للمصادقة
- Bcrypt لتشفير كلمات المرور
- قاعدة بيانات SQLite

### 5. 📊 لوحة تحكم تفاعلية
- عرض سجل الفحوصات
- رسم بياني لتطور الهيموجلوبين
- تذكيرات بالأدوية
- إحصائيات التحسن

### 6. 🥗 نظام التغذية الذكي
- خطط وجبات مخصصة
- دعم أنواع حمية متعددة
- مراعاة الحساسية الغذائية

### 7. 🔒 حماية الخصوصية
- حذف الصور تلقائياً بعد ساعة
- فحص جودة الصورة قبل التحليل
- تشفير البيانات الحساسة

## 🛠️ التقنيات المستخدمة

### Backend
- **Python 3.14**
- **Flask** - إطار عمل الويب
- **XGBoost** - نموذج الذكاء الاصطناعي
- **OpenCV** - معالجة الصور
- **SQLAlchemy** - إدارة قاعدة البيانات

### Frontend
- **HTML5/CSS3/JavaScript**
- **Chart.js** - الرسوم البيانية
- **Responsive Design** - تصميم متجاوب

### AI Model
- **Algorithm**: XGBoost Regressor
- **Features**: 34 color & texture features
- **Training Data**: 93 patients
- **Performance**: 
  - MAE: 1.48 g/dL
  - RMSE: 2.07 g/dL
  - CV MAE: 1.60 g/dL

## 🔬 كيفية عمل نموذج الذكاء الاصطناعي

### 1. استخراج الميزات (Feature Extraction)
من كل صورة، يتم استخراج 34 ميزة:

#### ميزات RGB (15)
- متوسط، انحراف معياري، وسيط لكل قناة
- الربع الأول والثالث

#### ميزات HSV (6)
- Hue (درجة اللون)
- Saturation (التشبع)
- Value (السطوع)

#### ميزات LAB (6)
- L (الإضاءة)
- A (محور أحمر-أخضر)
- B (محور أصفر-أزرق)

#### ميزات إضافية (7)
- نسبة اللون الأحمر
- مستوى التشبع
- نسب الألوان (R/G, R/B, G/B)
- سطوع Grayscale

### 2. التطبيع (Normalization)
جميع الميزات تُطبع باستخدام StandardScaler

### 3. التنبؤ (Prediction)
نموذج XGBoost يتنبأ بقيمة الهيموجلوبين بناءً على الميزات

### 4. التصنيف (Classification)
```python
Hb >= 12.0  → Normal
Hb >= 11.0  → Mild Anemia
Hb >= 8.0   → Moderate Anemia
Hb < 8.0    → Severe Anemia
```
